Description

The Certification course in Computer Applications & Multilingual DTP is a long term course offered by Techzone Academy for the students willing to learn in detail the various aspects of computers and the applications that run in a computer.

Syllabus

  • Module-I : Information Technology
  • Computer Fundamentals
  • Computer Organization
  • Operating System Concepts
  • Office Automation Tools (Libre Office)
  • IT Applications
  • Module-II : Business Systems
  • Fundamentals of Programming
  • Principles & Techniques of Programming
  • Introduction to Database (MySQL)
  • Accounting Fundamentals
  • Accounts Application (Tally)
  • Module-III : Web Application Development
  • Introduction to Internet
  • World Wide Web
  • Browsers, HTTP, FTP, Telnet
  • HTML5
  • Javascript, CSS3
  • Module-IV : Programming through ‘C’ & Python
  • Module – V : Multilingual DTP
  • English & Hindi Typing
  • Photoshop
  • Project Work : To be Submitted by the Student to the Trainer (File & CD)

    Min-Max Duration

    • Normal Duration – 1 Year
    • Extended Duration – 1.2 Year

    Exam Schedule

    • Module Exam – At the end of every module.
    • Quarterly Exam – Every three months.
    • Final Exam & Project – At the end of the course.
    • Note – Passing all the exams are compulsory. Three attempts will be provided to the students who fail the exams.

    Fees

    • Registration Fee – Rs. 250
    • Certification Fee – Rs. 300
    • Tution Fee – Rs. 600(Per Month)
